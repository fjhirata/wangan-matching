## 0. ひとことで

**「性格診断でタワマンの“棟”を提案」から、「診断＋条件で “いま実際に売り出し中の部屋” を提案する」サービスへ**大きく作り替えました。
実際の募集データ・成約データと連携し、**その場で買える具体的な住戸**と**実際の相場**が出るようになっています。

---

## 1. 何を変えたか（変更前 → 現在）

| 項目 | 変更前 | 現在 |
|---|---|---|
| 提案するもの | 建物（棟）のおすすめ | **いま募集中の住戸**（約1,072戸）から、建物TOP3 |
| 最初の入力 | 年収・年齢 | **予算・広さ・間取り・築年数・駅徒歩**（実際に探す条件） |
| 診断 | 5項目・20問・13タイプ | **6項目（眺望を追加）・24問・15タイプ**（六角形チャート） |
| 結果カード | 棟の紹介 | **建物単位の要約**（価格帯・募集坪単価の幅・成約相場・「相場以下の部屋あり」） |
| 部屋の詳細 | なし | 建物タップで**募集中の部屋を1戸ずつ一覧**（各部屋に相場比つき） |
| 坪単価 | 1種類 | **「募集価格」＋「実際の成約相場」を並べて割安/割高が分かる** |
| 再検索 | なし | 結果画面で**条件だけ変えてその場で再検索**（折りたたみ） |
| 配慮 | ― | 階数・面積は「30階台・70㎡台」と**ぼかし表示**（部屋の特定防止） |

## 2. データの取り方（2つのDBの“いいとこ取り”）

**すべて読み取り専用**。シートは一切書き換えません。

### ① いま売り出し中の部屋 →「★湾岸マンション募集DB ver2」
- 各エリアの募集マスタから、**最新の記録（現在2026/5/31）に価格がある部屋＝「現在募集中」**を抽出（約1,072戸／価格・広さ・階数・間取り・方位）。
- 募集価格は**常に最新のもの**を使用。

### ② 実際に売れた相場 →「旧・湾岸タワマンDB（成約5,843件・月1回更新）」
- 過去の成約から各建物の**「成約相場（坪単価）」**を算出。**49棟中48棟**をカバー。

### ③ 物件スペック →（別シート）
- 駅徒歩・築年・耐震・総戸数・共用施設。物件名で結合。

→ どのシートも更新されれば、**毎週（月曜朝）自動でアプリに反映**されます。

## 3. 提案ロジック・坪単価の考え方

### ① 物件を6つの観点で0〜100点に採点（すべて実データ）
| 観点 | 採点の根拠 |
|---|---|
| 資産性 | 過去成約の値上がり率 |
| 広さ | 募集住戸の面積 |
| 施設 | 共用施設（プール・ジム等）の充実度 |
| 立地 | 駅徒歩＋エリア |
| 居住性 | 築年・耐震・設備 |
| 眺望 | 階数 ÷ その建物の総階数（高層ほど高得点） |

- **マッチ度**＝利用者が重視する観点ほど重みを大きくして算出（診断結果に近い物件が上位に）。

### ② 坪単価は「募集」と「成約相場」を並べて表示（FJの強み）
結果カードは**建物単位**で表示します（特定の1部屋ではなく建物の全体像）。
- **募集価格帯**（例：1億2,780万〜1億4,980万円）… 条件に合う募集中住戸の価格の幅
- **募集坪単価の幅**（例：625〜805万/坪）… 同上、坪単価の幅
- **成約相場**（例：696万/坪）… その建物で**実際に売れた坪単価の直近3ヶ月平均**（取引が少ない棟は6→12ヶ月へ自動拡大）
- **「相場以下の部屋あり」** … 募集の中に成約相場より安い部屋があれば表示（＝お買い得サイン）
- **値上がり率**は「+128.7%(9年)」のように**通算年数付き**（データ期間の合計値。年利ではありません）

> ※募集価格は普通、実際の成約より高めに出ます。だから「募集 vs 成約相場」を並べると“本当にお得か”が分かる＝**実成約データを持つFJならではの見せ方**です。

### ③ 大事なポイント：すべて「あなたの条件に合う部屋だけ」で計算
- 価格帯・坪単価の幅・戸数は、**利用者が選んだ条件に合致する住戸のみ**から算出（例：1LDKを選べば1LDKの該当分だけ）。
- 条件に合う部屋が無ければ「**該当なし**」と正直に表示。
- 建物をタップして開く一覧も**同じ条件で絞り込み**（「全戸を見る」に切替も可能）。

## 4. 補足・現状の制約（正直な注記）

- 階数・面積は**ぼかし表示**（部屋の特定を避けるため）。正確な情報はLINE相談へ誘導。
- 新築1棟（THE TOYOMI TOWER）のみ成約・スペック未連携のため一部を募集価格で代替表示。
- 各スコア・坪単価は**実データに基づく“算出値”**（実測と推計を明示する方針）。
- 現状は**プロトタイプ／社内確認用のプレビュー**。表示データは2026/5/31時点。

## 5. さわり方（確認方法）
1. 上記URLを開く → 「無料で診断をはじめる」
2. 条件（予算・広さ・間取り 等）を選ぶ → 24問の診断に回答
3. 結果で「いま狙えるTOP3（建物）」と価値観チャート（6角形）を確認
4. 各建物カードをタップ → その建物の**条件に合う募集中の部屋**を一覧（相場比つき）
5. 「🔧 条件のみ変更して再検索」を開けば、回答そのままで条件だけ変えて再検索
