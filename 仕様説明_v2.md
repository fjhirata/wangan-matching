## 1. 今回、何を変えたか（ひとことで）

**「性格診断でタワマンの“棟”を提案」から、「診断＋条件で “いま実際に売り出し中の部屋” を提案」へ**進化させました。
（＝ポータルサイトのように、その場で買える具体的な住戸が出るようになりました）

### 主な変更点
| 項目 | 変更前 | 変更後 |
|---|---|---|
| 提案するもの | 建物（棟）のおすすめ | **いま募集中の住戸**（約1,072戸）からTOP3 |
| 最初の入力 | 年収・年齢 | **予算・広さ・間取り・築年数・駅徒歩**（実際に探す条件） |
| 診断 | 5項目・20問・13タイプ | **6項目（眺望を追加）・24問・15タイプ** |
| 坪単価の見せ方 | 1種類 | **「この部屋の募集坪単価」＋「建物の成約相場」＋割安/割高の判定** |
| 結果の操作 | 棟の紹介のみ | 各物件タップで**その棟の“条件に合う募集中の部屋”を一覧**／結果画面で**条件だけ変えて即再検索** |
| プライバシー配慮 | ― | 階数・面積は「30階台・70㎡台」のように**少しぼかして表示**（部屋の特定を防止） |

## 2. データの取り方（2つのDBの“いいとこ取り”）

**読み取り専用**で、2つのスプレッドシートから自動取得しています（シートは一切書き換えません）。

### ① いま売り出し中の部屋 →「★湾岸マンション募集DB ver2」
- 各エリアの募集マスタから、**最新の記録（現在2026/5/31）に価格がある部屋＝「現在募集中」**を抽出 → **約1,072戸**（価格・広さ・階数・間取り・方位）。
- 募集価格は**常に最新のもの**を使用（古い/取り下げは含めない）。

### ② 実際に売れた相場 →「旧・湾岸タワマンDB（成約5,843件・月1回更新）」
- 過去の成約データから各建物の**「成約相場（坪単価）」**を算出。**49棟中48棟**をカバー。

### ③ 物件スペック →（別シート）
- 駅徒歩・築年・耐震・総戸数・共用施設。物件名で結合。

→ どのスプレッドシートも更新されれば、**毎週（月曜朝）自動でアプリに反映**されます。

## 3. 点数（スコア）と坪単価の考え方

### ① 物件を6つの観点で0〜100点に採点（すべて実データ）
| 観点 | 採点の根拠 |
|---|---|
| 資産性 | 過去成約の値上がり率 |
| 広さ | 募集住戸の面積 |
| 施設 | 共用施設（プール・ジム等）の充実度 |
| 立地 | 駅徒歩＋エリア |
| 居住性 | 築年・耐震・設備 |
| 眺望 | 階数 ÷ その建物の総階数（高層ほど高得点） |

- **マッチ度**＝利用者が重視する観点ほど重みを大きくして算出（診断結果に近い物件が上位）。

### ② 坪単価は「募集」と「成約相場」を並べて表示（FJの強み）
- **この部屋 ○○万/坪** … その部屋の“いまの売り希望”価格
- **成約相場 △△万/坪** … その建物で**実際に売れた坪単価の直近3ヶ月平均**（旧タワマンDB。取引が少ない棟は6→12ヶ月へ自動拡大）
- **割安/割高の判定** … 募集が相場より安ければ「相場以下」🟢、同程度なら「ほぼ相場」、高ければ「相場より高め」
  - ※募集価格は普通、実際の成約より高めに出るため、この比較で“本当にお得か”が分かります＝**実成約データを持つFJならではの見せ方**。
- **値上がり率**は「+128.7%(9年)」のように**通算の年数付き**で表示（データ期間の合計値。年利ではありません）。

### ③ 絞り込み（必須条件）
- 予算・広さ・間取り・築年数・駅徒歩は**必須条件**として扱い、**合致する募集中の住戸だけ**を表示。合わなければ「該当なし」と正直に表示。
- 建物をタップして開く一覧も、**同じ条件で絞り込み**（条件に合う部屋だけ）。「全戸を見る」に切替も可能。

## 4. 補足・現状の制約（正直な注記）

- 階数・面積は**ぼかし表示**（部屋の特定を避けるため）。正確な情報はLINE相談へ誘導。
- 新築1棟（THE TOYOMI TOWER）のみ成約・スペック未連携のため一部を募集価格で代替表示。
- 各スコア・坪単価は**実データに基づく“算出値”**（実測と推計を明示する方針）。
- 現状は**プロトタイプ／社内確認用のプレビュー**。データは2026/5/31時点。

## 5. さわり方（確認方法）
1. 上記URLを開く → 「無料で診断をはじめる」
2. 条件（予算・広さ・間取り 等）を選ぶ → 24問の診断に回答
3. 結果で「いま狙えるTOP3」と価値観チャート（6角形）を確認
4. 各物件カードをタップ → その建物の“条件に合う募集中の部屋”一覧
5. 「🔧 条件のみ変更して再検索」を開けば、回答そのままで条件だけ変えて再検索
